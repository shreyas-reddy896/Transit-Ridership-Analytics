# Cloud-Based Transit Ridership Analytics Dashboard

## Overview
This project demonstrates a cloud-based data pipeline using AWS and Power BI to analyze transit ridership trends.

## Tech Stack
- AWS S3, RDS (PostgreSQL)
- Python (Pandas)
- Power BI

## Workflow
1. Generate dataset using Python
2. Store raw data in AWS S3
3. Load data into AWS RDS
4. Connect Power BI to visualize insights

## Insights
- Peak hours analysis
- Route-wise demand
- Delay patterns

## Future Improvements
- Real-time data streaming
- Predictive analytics
